charset

  • v::charset()

Validates if a string is in a specific charset.

v::charset('ASCII', 'açúcar'); //false
v::charset('ASCII', 'sugar');  //true
v::charset(array('ISO-8859-1', 'EUC-JP'), '日本国'); // true

The array format is a logic OR, not AND.
